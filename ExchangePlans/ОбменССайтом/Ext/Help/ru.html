План обмена с сайтом

План обмена Обмен с сайтом предназначен для записи и сохранения настроек обмена данными между информационной системой и сайтом (интернет магазином).

В программе Розница предусмотрен режим совместной работы розничного предприятия и интернет-магазина, при котором покупатель может заказать товар на сайте магазина и забрать покупку в одном из магазинов сети, указанном в качестве точки самовывоза. Оплата заказа производится на кассе магазина. Протокол обмена опубликован на сайте 1С.

Возможность обмена данными с сайтом Администратор может включить с помощью флага Обмен с сайтом по ссылке Администрирование - Синхронизация данных – Настрорйки синхонизации данных.

Создать новый узел обмена с сайтом можно с помощью кнопки Новая сихронизация данных в командной панели списка узлов.

Наименование узла обмена можно задать вручную. Если не заполнять поле Наименование, то при записи узла его значение будет сформировано автоматически.

Основные настройки

На странице Основные настройки необходимо выполнить следующие настройки:

· выбрать Режим обмена данными: Выгрузка товаров, Обмен Заказами;

· выбрать транспорт доставки данных: указать адрес сайта или локальный каталог для выгрузки. С помощью команды Проверить соединение... проверяется корректность задания адреса сайта.

Выполнение обмена

Обмен с сайтом может производиться автоматически по расписанию, заданному на странице Основные настройки, и в ручном режиме с помощью команды Выполнить обмен.

Выгрузка товаров

Страница Выгрузка товаров доступна, если в основных настройках установлен режим выгрузки товаров.

На странице указываются следующие параметры, необходимые для передачи данных по номенклатуре:

· Владелец каталога на сайте - юридическое лицо, оказывающие услуги интернет-торговли;

· Режим выгрузки товаров - выгружать все данные, удовлетворяющие настройке узла, или только измененные, то есть зарегистрированные в плане обмена;

· Выгружаемые данные - состав выгружаемых данных, в том числе признак выгрузки файлов изображений товаров;

· Точки самовывоза - список магазинов, в которых можно забрать товар.

В табличной части задается список выгружаемых на сайт каталогов. Список должен содержать как минимум один каталог. Если ни один каталог не задан, то он будет создан автоматически.

Обмен заказами

Страница Обмен заказами доступна, если в основных настройках установлен режим обмена заказами.

Администратор заполняет поля Организация и Причина отмены заказов по умолчанию. Значения этих полей нужны для регистрации заказов в системе при загрузке с сайта.

Также нужно задать параметры для создания новых позиций номенклатуры на тот случай, если при загрузке заказа с сайта в информационной базе не будут найдены соответствующие номенклатурные позиции.

В табличной части Администратор может установить соответствие статусов заказов на сайте и статусов заказов в информационной системе.
